Bem-vindo(a)! 👋  
Este repositório reúne **exemplos, exercícios e materiais de apoio** utilizados na formação **Software Developer** do **CESAE Digital – Lisboa**, com foco nos **princípios de Quality Assurance (QA)** e na criação de **testes unitários em Java utilizando JUnit 5**.

O objetivo é ajudar os formandos a compreender **a importância dos testes no desenvolvimento de software**, promovendo **boas práticas de qualidade, manutenção e fiabilidade do código**.

---

## 📚 Conteúdos Abordados

Ao longo dos exemplos e exercícios são explorados vários conceitos essenciais de **Quality Assurance**:

- 🧪 Introdução ao **Quality Assurance (QA)**
- 🔍 Importância dos **testes no ciclo de desenvolvimento**
- 🧩 **Testes unitários** em Java
- ⚙️ Configuração e utilização do **JUnit 5**
- ✔️ Escrita de **testes automatizados**
- 🧠 Estratégias para **testar métodos e lógica de negócio**
- 🚨 Testes de exceções e casos limite (*edge cases*)
- 📊 Boas práticas para **manter código testável**
